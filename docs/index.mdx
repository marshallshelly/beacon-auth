## Why BeaconAuth?

<CardGrid stagger>
  <Card title="Plugin Architecture" icon="puzzle">
    Extend functionality effortlessly. Only pay for what you use. Includes
    **OAuth** (GitHub, Google, Discord), **2FA** (TOTP + Backup Codes), and more
    out of the box.
  </Card>
  <Card title="Database Agnostic" icon="seti:db">
    Built-in support for **PostgreSQL**, **MySQL**, **SQLite**, **MSSQL**, and
    **MongoDB**.
  </Card>
  <Card title="Type-Safe Config" icon="setting">
    Leverage Go's strong typing for mistake-free configuration. Intuitive
    `WithOption` pattern for everything.
  </Card>
  <Card title="Framework Ready" icon="rocket">
    Native integrations for **Fiber**, **Chi**, **Gin**, **Echo**, **Gorilla
    Mux**, and standard **net/http**. Complete with multi-tenant support and
    session management.
  </Card>
</CardGrid>
